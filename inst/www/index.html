OpenCPU "European stocks" Application

Application créée à l'aide d'OpenCPU avec un template Sencha ExtJS.

Pour créer un tracé double cliquer sur une entreprise de la base de données à gauche. La courbe peut être modifié grâce aux boutons.

Si vous souhaitez créer un portefeuille, il faut entrer dans les actions du portefeuille (dans la barre en haut de cette fenêtre) tel que: AC.PA ACA.PA Il faut donc mettre les symboles des actions espacés d'un espace. Si vous voulez jouer sur les proportions mettez en pourcentage les proportions des actions que vous avez entrées, par exemple:75 25 Pour choisir le type de graphe faite dérouler les types de graphes qui est smoothplot par défault.
